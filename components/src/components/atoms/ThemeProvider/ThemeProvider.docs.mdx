## Element

Pass in an element selector or reference to change where the theme binds. By default, the theme attaches to the `:root` element.

```tsx live=true
<ThemeProvider
  defaultAccent="indigo"
  defaultMode={mode === 'dark' ? 'light' : 'dark'}
  element={previewRef.current}
>
  <Typography>
    The element mode is {mode === 'dark' ? 'light' : 'dark'} and accent is{' '}
    <Typography as="span" color="accent">
      indigo
    </Typography>
    .
  </Typography>
</ThemeProvider>
```

## Forced Theme

Passing `forcedAccent` and `forcedMode` allows you to override the internal state kept by `ThemeProvider`.

```tsx live=true
<ThemeProvider
  defaultAccent={accent}
  defaultMode={mode}
  element={previewRef.current}
  forcedAccent="indigo"
  forcedMode="dark"
>
  <Stack space="2">
    <Typography>
      Parent mode is {mode} and element mode is forced to dark.
    </Typography>
    <Typography>
      Parent accent is{' '}
      <Typography as="span" color={accent}>
        {accent}
      </Typography>{' '}
      and element accent is forced to{' '}
      <Typography as="span" color="accent">
        indigo
      </Typography>
      .
    </Typography>
  </Stack>
</ThemeProvider>
```
