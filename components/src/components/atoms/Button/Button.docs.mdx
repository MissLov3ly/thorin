## States

```tsx live=true
<div>
  <Button loading>Loading</Button>
  <Button disabled>Disabled</Button>
  <Button
    disabled={getState('loading')}
    loading={getState('loading')}
    onClick={() => {
      toggleState('loading')
      setTimeout(() => toggleState('loading'), 1000)
    }}
  >
    Click Me
  </Button>
</div>
```

## Tones

When `variant` is set to `primary` or `secondary`, you can `tone` to change the button accent color.

```tsx live=true
<div>
  <div>
    <Typography variant="label">Primary</Typography>
    <div>
      <Button tone="accent" variant="primary">
        Accent
      </Button>
      <Button tone="blue" variant="primary">
        Blue
      </Button>
      <Button tone="green" variant="primary">
        Green
      </Button>
      <Button tone="red" variant="primary">
        Red
      </Button>
```

## Affixes

Optional `center` prop reserves space for affixes and loading spinner.

```tsx live=true
<div>
  <div>
    <Typography variant="label">Default</Typography>
    <div>
      <Button prefix={<div>plus</div>}>Add Entry</Button>
      <Button suffix={<div>plus</div>}>Add Entry</Button>
      <Button prefix={<div>plus</div>} suffix={<IconPlusSmall />}>
        Add Entry
      </Button>
      <Button loading prefix={<IconPlusSmall />}>
        Add Entry
      </Button>
    </div>
  </div>

  <div>
    <Typography variant="label">Center</Typography>
    <div>
      <Button center prefix={<IconPlusSmall />}>
        Add Entry
      </Button>
      <Button center suffix={<IconPlusSmall />}>
        Add Entry
      </Button>
      <Button center prefix={<IconPlusSmall />} suffix={<IconPlusSmall />}>
        Add Entry
      </Button>
      <Button center loading prefix={<IconPlusSmall />}>
        Add Entry
      </Button>
    </div>
  </div>
</div>
```
