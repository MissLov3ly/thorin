## Usage

Select "Open in Playroom" to get started with the example below.

```tsx live=true
<div style={{ display: 'flex' }}>
  <Button
    disabled={getState('loading')}
    loading={getState('loading')}
    prefix={<LockClosedSVG />}
    tone="blue"
    variant="primary"
    onClick={() => {
      toggleState('loading')
      setTimeout(() => toggleState('loading'), 1000)
    }}
  >
    Connect Wallet
  </Button>
</div>
```

With Playroom, you can test different screen sizes and themes, share links to prototypes, and use all system components without needing to import anything.

## State and Interactivity

You can control Playroom state and interactivity using the variables and functions that are injected into the global context:

```ts
// State
setDefaultState(key, value)
getState(key)
setState(key, value)
toggleState(key)
resetState()
// Theme
accent
mode
setAccent(accent)
setMode(mode)
// CSS
vars
```

```tsx live=true
<div style={{ display: 'flex' }}>
  <Typography size="extraLarge">
    {getState('counter') != null ? getState('counter') : 2}
  </Typography>
  <div
    style={{
      display: 'flex',
      marginLeft: '24px',
      flexGap: '12px',
      gap: '12px',
      flexAlign: 'center',
      flexDirection: 'row',
    }}
  >
    <Button
      disabled={getState('locked')}
      size="small"
      variant="tertiary"
      onClick={() => setState('counter', getState('counter') - 1)}
    >
      Decrement
    </Button>
    <Button
      disabled={getState('locked')}
      size="small"
      variant="tertiary"
      onClick={() => setState('counter', getState('counter') + 1)}
    >
      Increment
    </Button>

    {getState('counter') != null && getState('counter') !== 2 && (
      <Button
        disabled={getState('locked')}
        size="small"
        variant="tertiary"
        onClick={() => resetState()}
      >
        Reset
      </Button>
    )}

    <Button
      size="small"
      variant="tertiary"
      onClick={() => toggleState('locked')}
    >
      {getState('locked') ? 'Unlock' : 'Lock'}
    </Button>

    <VisuallyHidden>{setDefaultState('counter', 2)}</VisuallyHidden>
  </div>
</div>
```
