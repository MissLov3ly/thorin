## States

```tsx live=true
<Stack flexDirection="row" flexWrap="wrap">
  <Button loading>Loading</Button>
  <Button disabled>Disabled</Button>
  <Button
    disabled={getState('loading')}
    loading={getState('loading')}
    onClick={() => {
      toggleState('loading')
      setTimeout(() => toggleState('loading'), 1000)
    }}
  >
    Click Me
  </Button>
</Stack>
```

## Tones

When `variant` is set to `primary` or `action`, you can `tone` to change the button accent color.

## Affixes

Optional `center` prop reserves space for affixes and loading spinner.

```tsx live=true
<Stack gap="64px">
  <Stack>
    <Typography variant="label">Default</Typography>
    <Stack flexDirection="row" flexWrap="wrap">
      <Button prefix={<PlusSmallSVG />}>Add Entry</Button>
      <Button suffix={<PlusSmallSVG />}>Add Entry</Button>
      <Button prefix={<PlusSmallSVG />} suffix={<PlusSmallSVG />}>
        Add Entry
      </Button>
      <Button loading prefix={<PlusSmallSVG />}>
        Add Entry
      </Button>
    </Stack>
  </Stack>

  <Stack>
    <Typography variant="label">Center</Typography>
    <Stack flexDirection="row" flexWrap="wrap">
      <Button center prefix={<PlusSmallSVG />}>
        Add Entry
      </Button>
      <Button center suffix={<PlusSmallSVG />}>
        Add Entry
      </Button>
      <Button center prefix={<PlusSmallSVG />} suffix={<PlusSmallSVG />}>
        Add Entry
      </Button>
      <Button center loading prefix={<PlusSmallSVG />}>
        Add Entry
      </Button>
    </Stack>
  </Stack>
</Stack>
```
