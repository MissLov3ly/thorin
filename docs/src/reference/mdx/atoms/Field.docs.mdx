---
title: Field
description: Wrapper for form content that adds labels and status messages
---

```tsx
import { Field } from '@ensdomains/thorin'
```

```tsx live=true expand=true

  <div style={{ backgroundColor: 'black', height: 8 }} />
</Field>
```

## Consume ID

The content wrapped by field sets an `id` automatically for accessibility. You can customize the behavior by using the render prop.

```tsx live=true
<DeleteMe>
  <Field label="Automatically injected">
    <div style={{ backgroundColor: 'black', height: 8 }} />
  </Field>

  <Field label="Render prop">
    {(ids) => (
      <div style={{ backgroundColor: 'black', height: 8 }} {...ids.content} />
    )}
  </Field>
</DeleteMe>
```
